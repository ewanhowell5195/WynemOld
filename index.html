Privacy Policy

By using the discord bot Wynem, you agree that you have read and agree to this policy.

This is our Privacy Policy, which will describe the types of data collected, how it is used, how to opt out of data collection, and how to remove your data.

The terms "we", "us", and "our" refer to the owner and moderators of Wynem. The terms "bot" and "Wynem" refer to the Discord bot itself.

We may update this Privacy Policy at any time. Any changes that are made will be made effective immediately. If you are a regular user of Wynem, please make sure you are kept up to date with this Privacy Policy.


PRIVACY STATEMENT

We respect the privacy of your information. This policy describes the types of information that we may collect from you or that you may provide when you use Wynem, as well as describing our practices for collecting, using, maintaining, protecting and disclosing that information.

This policy applies to any data collected through these sources:

- Data provided directly by users

- Direct messages

- Servers the bot is added to

- Any commands ran


THE INFORMATION WE COLLECT AND HOW WE COLLECT IT

Personally Identifiable Information:

Wynem collects and stores Minecraft usernames that you tie to your Discord account. These are provided voluntarily and can be removed by the user at any time. This will only be personally identifiable if the username is personally identifiable. These are stored locally on the bot server.

Servers:

Whenever Wynem joins and leaves a server, a log is stored within a private Discord channel which lists the details of the server, such as name, member count, date created, and number of channels. This data is only stored within Discord and is never stored locally.

Server logs may be enabled in any server that Wynem is in. Server logs log your activity in a server. The data logged includes edited message content, deleted message content, nicknames, voice channels joined, and any moderation taken against you. This data is only stored in the server the specific activity occurred in.

When a member is muted or warned, a moderation log is saved in a local file on the bot server. Only the 10 newest warn logs and 10 newest mute logs are stored. The total count is still tracked however.

Direct Messages:

When you send a direct message to Wynem, this message is sent to a private Discord channel. This logs the full message content, including attachments, except for stickers and emojis that Wynem does not have access to. This data is only stored within Discord and is never stored locally.

Commands:

Any command executed through Wynem is sent to a private Discord channel. This logs the full message content, including attachments. This data is only stored within Discord and is never stored locally.


HOW WE USE INFORMATION

Servers:

Server join/leave information is logged to keep track of the kinds of servers that Wynem is being used in, as well as the frequency that Wynem is being added/removed from servers.

Direct Messages:

Direct messages are logged so that it is possible to provide support through direct messages.

Commands:

Commands are logged to help track error messages and bot abuse. When an error occurs, the command logs can be used to see the activity that occurred around the error.


OPTING OUT

How to opt out of having this data collected:

Server owners/admins:

To opt out of having any commands logged from your server, remove Wynem from your server.

Server members:

To opt out of having your commands logged from a server using Wynem, leave the server.

Users:

To opt out of having direct messages logged, do not direct message the bot.


REMOVAL OF DATA

Some commands exist for you to remove your own data, such as removing Minecraft usernames. These commands can be used to remove that data.

Attachments:

Any attachments logged are only ever logged using the original Discord URL. By deleting the message containing the attachment, you are deleting the content from Discord's servers, therefore the attachment URL will no longer be valid.

Commands and Direct Messages:

To remove any command and direct messages that have been logged, please contact me through the contacts listed at the bottom of this policy, with the Message and User ID of the log you would like removed.

Server Logs and Moderation Records:

We do not own any of the data logged about you through server logs and moderation records. This data is owned by the Server owners/admins. You will have to contact them to have your data removed.


CHILDREN'S PRIVACY

Discord does not provide services to children under the age of 13, so this extends to the usage of Wynem. If you are a parent or guardian and you are aware that your child has provided us with personal data, please contact us so that we can remove it.


THIRD-PARTY WEBSITES

Wynem may link to third-party websites. These websites are separate from Wynem and will have their own privacy policies. Any data collected through these third-party websites is separate and unrelated to Wynem.
